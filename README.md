# 賓果賓果分析工具

台灣彩券賓果賓果（Bingo Bingo）開獎資料分析工具，提供熱度圖、號碼排行、共現分析等功能。

## 環境需求

- [Node.js](https://nodejs.org/) v16 以上

## 安裝

```bash
# 複製或下載專案後，進入專案目錄
cd bingo

# 無需安裝相依套件（僅使用 Node.js 內建模組）
```

## 啟動

```bash
# 一般啟動
npm start

# 開發模式（檔案變更自動重啟，需 Node.js v18+）
npm run dev
```

啟動後開啟瀏覽器前往：

```
http://localhost:8081
```

## 功能說明

| 功能 | 說明 |
|------|------|
| 熱度圖 | 顯示各號碼在指定期數內的出現頻率 |
| 號碼排行 | 依出現次數排列熱門與冷門號碼 |
| 分布熱圖 | 以矩陣呈現每期各號碼的開獎分布 |
| 共現分析 | 查詢選定號碼與其他號碼同期出現的機率 |
| 對獎查詢 | 輸入期數區間與號碼進行對獎 |

## 快取機制

開獎資料會自動快取於 `cache/` 目錄下（保留 5 分鐘），減少重複呼叫台灣彩券 API。
